Filhos do Tipo Array

Filhos do Tipo Array

Prover um array como filho de um componente é muito comum. É assim que listas são desenhadas em React.

Nós usamos map() para criar um vetor de elementos React para cada valor no array (ou vetor).

<ul>
  {["first", "second"].map(item => (
    <li>{item}</li>
  ))}
</ul>

Isso é equivalente à prover um array literal.

<ul>{[<li>first</li>, <li>second</li>]}</ul>

Veja o exemplo deste repositório de teste do react-bootstrap: https://github.com/crocarneiro/react-bootstrap-test/blob/master/src/App.js.

Este padrão pode ser combinado com desestruturação de props, propagação de atributos JSX, e outros componentes, para fazer algo profissional.

<ul>
  {arrayOfMessageObjects.map(({ id, ...message }) => (
    <Message key={id} {...message} />
  ))}
</ul>
