Renderização Condicional

Renderização Condicional

Você não pode utilizar comandos if/else dentro da declaração de um componente, portanto, o operador condicional (ternário) e a avaliação de curto-circuito (short-circuit) são seus amigos.

if

{
  condition && <span>Rendered when `truthy`</span>;
}

unless

{
  condition || <span>Rendered when `falsy`</span>;
}

if-else

{
  condition ? (
    <span>Rendered when `truthy`</span>
  ) : (
    <span>Rendered when `falsy`</span>
  );
}
