Atribuição via Desestruturação

Atribuição via desestruturação é uma característica do Javascript que foi adicionada à linguagem no ES2015, então, pode não parecer muito familiar. Pense nela como o oposto da atribuição literal.

let person = { name: "chantastic" };
let { name } = person;

Funciona com vetores também.

let things = ["one", "two"];
let [first, second] = things;

Atribuição via desestruturação é muito usada in funções componentes. Essas declarações de componentes abaixo são equivalentes:

function Greeting(props) {
  return <div>Hi {props.name}!</div>;
}

function Greeting({ name }) {
  return <div>Hi {name}!</div>;
}

Existe uma sintaxe para coletar as props restantes para dentro de um objeto, ela é chamada de sintaxe de parâmetros rest e se parece com isso:

function Greeting({ name, ...restProps }) {
  return <div>Hi {name}!</div>;
}

Aqueles três pontos (...) pegam todas as propriedades restantes e atribuem-nas ao objeto restProps.
